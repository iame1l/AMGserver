关于龙虎斗修改筹码的注意点

1.龙虎斗筹码是在客户端控制

```C++
const  int  BET_ARES = 3;    //3个下注区域  0龙  1虎   2 和
  const  int  CHOUMANUM = 6;    //6个筹码区域
  const  LLONG  G_i64ChouMaValues[CHOUMANUM] = { 1000, 10000, 50000, 100000, 500000,1000000 };//6个筹码值
```



2.机器人需要另外对应上客户端的筹码

```C++
AI的对应

const	__int64	G_i64ChouMaValues[CHOUMA_NUM] = {1000,10000,50000,100000,500000,1000000};
```
